# 우주 테마 전환 (Day 1) 구현 계획

**목표:** 통통 디펜스를 우주 테마로 — 볼·몬스터·블록·배경·스킬명 교체. 코드 로직은 불변, 에셋·데이터만 교체.

**검증 방식:** Unity 에셋/씬 작업이라 TDD 부적합 → 컴파일 선확인(내가) + Play 체크리스트(유저). 코드 테스트는 무한모드·보스(Day 3-4)에서.

**설계 근거:** docs/2026-07-10-우주-리메이크-설계.md

## Global Constraints
- 게임 로직 코드 변경 금지 (에셋·CSV·씬 배선만)
- 볼 스프라이트만 Resources 폴더 필요 (Resources.Load 경로 기반)
- 몬스터·배경은 씬 SerializeField 참조라 Art/Space 그대로 사용 가능
- 에셋 원본: Builds/_kenney_보관/, 작업본: Assets/Art/Space/

**스텝:**
