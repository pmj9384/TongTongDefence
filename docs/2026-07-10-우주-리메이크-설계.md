# 우주 핀볼 디펜스 — 리메이크 설계 (v1)

> 2026-07-10 브레인스토밍 확정. 통통 디펜스 채용 과제작을 "내 게임"으로 발전.
> (게임명 가제 — 테마 확정 후 재명명)

## 1. 목표
- 1차: 포트폴리오 완성작 — "내가 기획한 우주 테마 핀볼 디펜스"
- 2차(잘되면): 스토어 출시 — 무한 모드 + AdMob 수익화
- 기간: 1주 집중 개발 → 포폴 패키징 → (출시 결심 시) 스토어 절차 3~4일

## 2. 핵심 변경 3가지

### ① 테마 전환 (숲 → 우주) — 코드 거의 안 건드림
- 볼 = 에너지 탄/플라즈마 / 몬스터 = 외계 생명체·적 함선
- 필드 = 우주 정거장 격자 / 캐릭터 = 파일럿·포탑
- 배경 = 별+성운 (코드 생성 — 스프라이트 불필요)
- 스킬 이름 = SF로 (SkillTable.csv 텍스트만)

### ② 유한 → 무한 모드 — WaveManager 개조
- 지금: 20행 소진 → 클리어(끝) / 변경: 웨이브 계속 이어지고 난이도 점증(HP·속도↑), 끝 없음
- 목표 = 점수(처치 수)·최고기록 경쟁

### ③ 보스 (주기적 관문) — 기존 몬스터 코드 재활용
- 20웨이브마다 보스 등장 → 격파하면 다음 구간
- 형태 = "거대 체력 먹통이" (기존 몬스터 크게 + 체력 왕창). 신규 시스템 최소
- 패턴 공격은 나중 욕심 (v1 제외)

## 3. 확정 세부
- 스킬: 3레벨 상한 유지. 만렙 후 성장은 나중 고민 (v1은 그대로)
- 점수: 처치 수 (PlayerLevel.TotalKills 재활용 — 공짜)
- 보스 주기: 20웨이브마다

## 4. 에셋 · 수익화
- 아트: Kenney 우주 팩 (CC0) — 볼·적·이펙트·UI. 상업 이용 자유
- 사운드: Kenney 오디오 (CC0)
- 수익화(출시 시): AdMob 전면(게임오버 가끔) + 리워드(부활). 원작 "광고 부활"을 실제 구현

## 5. 1주 스코프 (Day별)
- Day 1-2: 테마 교체 — Kenney 에셋 + 스프라이트/CSV 이름 교체 + 우주 배경 코드
- Day 3-4: 무한 모드(WaveManager) + 보스/구간 반복 + 점수 시스템
- Day 5: 사운드(Kenney) + 폴리시
- Day 6-7: 포폴 패키징 (플레이 영상·README·회고)

## 6. 재활용 vs 신규
- 재활용(거의 그대로): 볼 물리·인벤토리, 스킬 시스템(CSV), 행 컨베이어, 순수 코어+테스트 42, 오브젝트 풀, UI 프레임
- 신규/개조: 무한 웨이브(WaveManager), 보스(타입+주기 스폰), 점수 시스템, 우주 배경, AdMob(출시 시)
- 버림: 유한 클리어 로직, 숲 에셋
