# 통통 디펜스 모작 — 한 장 요약

핀볼 디펜스: 볼을 쏘면 물리로 튕기며 몬스터를 때리고, 바닥에 튕겨 돌아오면 재발사. 몬스터는 위에서 줄지어 내려오고, 다 잡으면 클리어.

## 구조 (Unity 6 / C#)

- **매니저 8개 + 중앙 GameManager(상태 머신)** — 싱글톤 대신 GameManager가 각 매니저를 프로퍼티로 들고 주입. 매니저끼리는 C# 이벤트로 통신
- **규칙은 순수 C#으로 분리** (데미지 공식, 카드 뽑기 규칙, 볼 회수 순서 등) → 유니티 없이 도는 테스트 42개
- **수치는 전부 CSV** — 스킬 테이블 + 스테이지 몬스터 배치 패턴. 코드 안 고치고 밸런싱
- **오브젝트 풀 전면 사용** (볼/몬스터/데미지 텍스트) + 대량 전투 대비 사전 워밍

## 핵심 로직 3개

1. **볼 인벤토리** — 보유한 볼 하나하나가 개체. 큐에 넣고, 먼저 회수된 볼부터 재발사 (쿨다운 없음 — 회수가 곧 발사 자원)
2. **데미지 소스 태그** — 모든 데미지에 "누가 때렸는지"를 실어 보냄 (볼 직격/화상/레이저/파편/폭발). 집계 매니저가 스킬별로 모아서 전투 통계 창(누적/DPS/비율)에 표시
3. **행 컨베이어 스폰** — 웨이브를 묶음이 아니라 "줄"로: 시작 5줄, 이후 앞줄이 한 칸 내려올 때마다 다음 줄 등장. 박자를 시간이 아니라 거리로 정의해서 줄 간격이 항상 격자에 맞음. 세로 2칸/가로 2칸 몬스터도 이 격자 위에서 배치

## 스킬 (10종 × 3레벨)

액티브 볼 5 (화염 도트/빙결 감속/레이저 행 전체/유령 관통/폭발 파편) + 패시브 5 (노멀 강화, 벽 튕김 비례 데미지, 앞·뒤면 치명타, 처치 시 폭발). 레벨업(킬 수)마다 3장 중 1장 선택.

## 기타

- 필드 격자는 배경 그림 기준으로 계산 → 어떤 화면비에서도 그림과 안 어긋남
- UI는 씬에서 조립(Inspector), 코드는 로직만. 반복 조립은 에디터 메뉴 도구로
- APK: IL2CPP + ARM64/ARMv7 (64비트 폰 포함 전부 설치 가능)
